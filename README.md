# Myntra Home Page Clone 🛍️

A responsive and visually accurate clone of the **Myntra** landing page, built using **HTML5** and **CSS3**. 
This project focuses on replicating the UI/UX of India's leading fashion e-commerce platform, featuring a functional header, promotional banners, and category grids.
 
## ✨ Features
- **Sticky Navigation:** A fixed header with a clean, branded navigation bar.
- **Search Bar UI:** Integrated with Google Material Symbols for a professional look.
- **Interactive Links:** Hover effects on navigation items and category images.
- **Flexbox Layout:** Modern CSS layout techniques for alignment and spacing.
- **Brand Typography:** Uses the 'Assistant' font family to match the official Myntra aesthetic.

## 🛠️ Technologies Used
- **HTML5:** Semantic structure for the web page.
- **CSS3:** Custom styling, Flexbox for layouts, and hover transitions.
- **Google Material Symbols:** Icons for Profile, Wishlist, and Bag.
- **Google Fonts:** For high-quality web typography.
